## Quickstart

Get started with Mantis docs, by building and running them with the following commands:

```
$ git clone git@github.com:input-output-hk/mantis-docs.git
$ cd mantis-docs
$ npm install
$ npm start
```

Visit `http://localhost:8000/` to view the app.
